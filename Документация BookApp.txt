# Официальная документация приложения BookApp
Обзор
BookApp - это настольное приложение, разработанное для управления коллекцией книг. Оно позволяет пользователям добавлять информацию о книгах в базу данных, а также просматривать и обновлять список книг.
Требования
- Java Runtime Environment (JRE) версии 8 или выше.
- MySQL Server для хранения данных о книгах.
Установка и запуск
1. Убедитесь, что JRE и MySQL Server установлены и настроены.
2. Скомпилируйте исходный код приложения с помощью JDK.
3. Запустите скомпилированный файл `.jar` для открытия интерфейса приложения.
Использование приложения
 Главный интерфейс
При запуске приложения отображается главное окно с полями для ввода данных о книге и кнопками для управления данными.
 Добавление книги
1. Введите название и автора книги в соответствующие поля.
2. Нажмите кнопку "Выбрать файл" для указания файла книги.
3. Нажмите "Добавить книгу" для сохранения информации в базу данных.
Просмотр и обновление списка книг
- Нажмите "Обновить список" для просмотра актуального списка книг.
Конфигурация базы данных
Для работы приложения необходимо настроить соединение с базой данных MySQL, указав правильные учетные данные и параметры подключения в коде.
